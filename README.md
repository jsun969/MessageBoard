# MessageBoard
A simple message board with PHP and MDUI  
一个简单的PHP+MDUI留言板

首先请导入**message.sql**至数据库

然后创建**config.php**
```php
<?php
// 网站标题
const title = "荆棘留言板";
// 数据库地址
const db_host = "233.233.233.233";
// 数据库名
const db_name = "name";
// 数据库用户名
const db_username = "username";
// 数据库密码
const db_password = "password";
// 数据库端口
const db_port = 3306;
```
